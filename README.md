Project Description:
This is a web development backend project of a car dealership. This site allows users to browse a selection of vehicles, view information about each car, and can also submit their car in for a service.
This application is for customers looking to buying or look at vehicles. It is also for the staff to manage the listing of vehicles and service requests.
The goal of this project is to demonstrate my skills in backend web development ove the course of this semester.
